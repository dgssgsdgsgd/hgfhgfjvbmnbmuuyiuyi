# Expense Tracker

**Автор:** [Фирулёва Наталья Сергеевна]

**Дата создания** [25.04.2026]

## Описание

Expense Tracker — это десктопное приложение для учёта личных расходов

### Возможности:
* добавлять расходы с указанием суммы, категории и даты;

* просматривать все записи в таблице;

* фильтровать записи по категории и периоду;

* подсчитывать общую сумму расходов за выбранный период;
